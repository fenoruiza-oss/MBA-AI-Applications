# Eval Summary

This project includes explicit evaluation fixtures for the deterministic router, the strategic-fit agent, the markdown report writer, and the end-to-end pipeline.

- `router.jsonl`: validates threshold behavior and reason-code assignment.
- `strategic-fit.jsonl`: validates theme mapping and conservative recommendation behavior.
- `report-writer.jsonl`: validates required markdown sections.
- `pipeline-e2e.jsonl`: validates overall outcome for representative targets.

These datasets are intentionally small and inspectable for the assignment. They are meant to demonstrate evaluation discipline rather than production-grade coverage.
